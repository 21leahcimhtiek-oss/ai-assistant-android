# TraumaHeal - PTSD & Trauma-Focused Therapy App

## Overview

TraumaHeal is a specialized mental health app designed for individuals dealing with PTSD, complex trauma, and trauma-related disorders. It provides evidence-based trauma therapy techniques including EMDR, Cognitive Processing Therapy (CPT), and trauma-informed grounding exercises.

## Key Features

### 1. Trauma-Focused AI Therapist
- Specialized in PTSD and trauma processing
- Uses evidence-based approaches: EMDR, CPT, PE (Prolonged Exposure)
- Trauma-informed communication style
- Crisis detection and safety planning

### 2. Trauma Timeline Tracking
- Visual timeline of traumatic events
- Trigger identification and logging
- Trauma anniversary tracking
- Impact assessment over time

### 3. Grounding & Dissociation Tools
- 5-4-3-2-1 sensory grounding technique
- Butterfly tapping (bilateral stimulation)
- Grounding object identification
- Dissociation detection and response

### 4. EMDR Simulation
- Bilateral stimulation with visual/auditory cues
- Eye movement tracking simulation
- Desensitization exercises
- Memory processing support

### 5. Trauma Processing Exercises
- Imaginal exposure techniques
- Narrative therapy exercises
- Thought records for trauma processing
- Emotional regulation tools

### 6. Safety Planning
- Crisis safety plan creation
- Emergency contacts
- Coping strategies for flashbacks
- Hospitalization resources

### 7. Professional Therapist Integration
- Connect with trauma-specialized therapists
- Share trauma timeline with therapist
- Video teletherapy for trauma sessions
- Treatment progress tracking

## Technical Stack

- React Native with Expo
- TypeScript
- AsyncStorage for local data
- OpenRouter AI for trauma-focused therapy
- HIPAA-compliant data encryption

## Data Privacy

All trauma data is encrypted locally on the device. No data is sent to servers without explicit user consent. Users can export or delete all data at any time.
