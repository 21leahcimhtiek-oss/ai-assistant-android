# TraumaHeal - Design Specification

## Screen List

1. **Onboarding** - Welcome screen with trauma-informed messaging
2. **Home Dashboard** - Daily check-in, trauma triggers, safety status
3. **Trauma Chat** - AI therapist specialized in trauma processing
4. **Trauma Timeline** - Visual timeline of traumatic events and triggers
5. **Grounding Tools** - Quick access to dissociation management techniques
6. **EMDR Simulation** - Bilateral stimulation exercises
7. **Safety Plan** - Crisis safety plan and emergency contacts
8. **Therapist Marketplace** - Connect with trauma-specialized therapists
9. **Progress Tracking** - Recovery milestones and healing progress
10. **Settings** - Preferences, data management, crisis resources

## Primary Content and Functionality

**Home Dashboard**: Daily trauma check-in (0-10 scale), current triggers, grounding status, quick access buttons to safety tools, upcoming therapy sessions, and motivational affirmations for trauma survivors.

**Trauma Chat**: AI therapist trained in trauma-informed care, EMDR techniques, CPT, and prolonged exposure therapy. Conversation history with trauma-specific prompts like "What triggered you today?" and "How are you managing flashbacks?"

**Trauma Timeline**: Visual chronological display of traumatic events (user-created), trigger identification, symptom tracking, and recovery milestones. Color-coded by trauma type or severity.

**Grounding Tools**: 5-4-3-2-1 sensory technique, butterfly tapping bilateral stimulation, grounding object identification, and dissociation response protocols.

**EMDR Simulation**: Bilateral visual/auditory stimulation with eye movement tracking, desensitization exercises, and memory processing support.

**Safety Plan**: Crisis safety plan creation, emergency contact list, coping strategies for flashbacks, and hospitalization resources.

## Key User Flows

**Flow 1 - Daily Check-in to Grounding**
1. User opens app → Home dashboard
2. Completes daily trauma check-in (mood, triggers, symptoms)
3. If dissociation detected → Suggest grounding tool
4. User selects 5-4-3-2-1 technique
5. Guided through sensory grounding exercise
6. Return to home with improved status

**Flow 2 - Trauma Processing Session**
1. User opens Trauma Chat
2. AI therapist asks about current triggers
3. User describes traumatic memory
4. AI guides through EMDR bilateral stimulation
5. User processes memory with eye movement tracking
6. Session summary and coping strategies provided

**Flow 3 - Crisis to Safety**
1. User experiences flashback or crisis
2. Taps crisis button on any screen
3. Safety plan displayed with emergency contacts
4. Grounding exercises offered
5. Option to call crisis line or contact therapist
6. Crisis log recorded for therapist review

## Color Choices

**Primary Colors** (Calming, trauma-informed):
- Deep Blue (#1E3A8A) - Trust, safety, stability
- Soft Teal (#0D9488) - Healing, growth, calm
- Warm Beige (#F5F3FF) - Comfort, safety

**Accent Colors**:
- Gentle Green (#10B981) - Recovery, progress
- Soft Orange (#F97316) - Warmth, support
- Muted Purple (#8B5CF6) - Introspection, processing

**Status Colors**:
- Safe Green (#22C55E) - Grounded, stable
- Warning Amber (#F59E0B) - Triggered, alert
- Crisis Red (#EF4444) - Emergency, immediate help needed

## Typography

- Headlines: Bold, clear, trauma-informed language (avoid triggering terms)
- Body Text: Easy to read, high contrast for accessibility
- Labels: Clear, simple language for crisis situations

## Interaction Design

- Large, accessible buttons for crisis situations
- Gentle animations (no jarring transitions)
- Haptic feedback for grounding exercises
- Voice guidance option for dissociated states
- Dark mode for light sensitivity
- High contrast for accessibility
