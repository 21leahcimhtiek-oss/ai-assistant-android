# Ai Assistant Android

> Unrestricted AI coding assistant with complete phone access, web scraping with rotating proxies, and deep web connectivity · Built with Manus



## Features

- 🤖 AI-powered core features via OpenRouter
- 🔐 Authentication with Supabase
- 💳 Payments with RevenueCat
- 📱 React Native / Expo mobile app
- 🎨 Modern UI with Tailwind CSS

## Quick Start

```bash
git clone https://github.com/21leahcimhtiek-oss/ai-assistant-android
cd ai-assistant-android
cp .env.example .env.local
npm install
npm run dev
```

## Environment Variables

See `.env.example` for all required environment variables.

## Tech Stack

- React Native / Expo
- Supabase (auth + database)
- OpenRouter (AI)
- RevenueCat (subscriptions)
- TypeScript
